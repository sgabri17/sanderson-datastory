My Video Story

Tinder is not as private as people think it is. Tinder has a location-based API and an API is an application program interface. There are two groups of APIs: a public API and a private API. Tinder's API is public, meaning that the data is open for other developers to use. One app in particular, Swipebuster, takes the data from Tinder and sells it to customers that want to track other Tinder users. Even though it sounds like an invasion of privacy, all of the data on Tinder is seen as public information by the app and can be revealed through the public API. Despite the fact that Swipebuster users use the app to target cheating partners, the developer stated that he created the app to highlight oversharing online. In conclusion, through my data discovery I determined that the public should be more cautious with what they post online, because it is never clear who will see it later.
